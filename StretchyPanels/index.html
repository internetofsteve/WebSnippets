Simple panel components that use flexbox to maintain a consistent height

50% Fixed-Width Panel

You can fix the width of a panel via the .panel-container class.

These two panels have a width of 50%, minus their left and right margins (achieved using calc()).

If you do not set a width and are planning on using flex-grow, you will likely see some weird behavior as the width of your content will be used to calculate the rate of flex-grow.

You can make things more flexible by setting flex-basis instead of width, it depends on your needs.
